# Architecture Decision Records (ADR)

Use this folder to preserve major architectural and musical-system decisions.

## Naming

- `ADR-0001-title.md`
- `ADR-0002-title.md`

## Template

```md
# ADR-XXXX: <Title>

## Status
Accepted | Proposed | Superseded

## Context
What problem or pressure led to this decision?

## Decision
What is the chosen approach?

## Alternatives Considered
- Option A
- Option B

## Consequences
What tradeoffs does this introduce?

## Follow-ups
Any required implementation or docs updates.
```
